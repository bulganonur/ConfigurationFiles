# KDE config file paths

#### Panel
* `.config/plasma-org.kde.plasma.desktop-appletsrc`

#### Appearance & Style
* Fonts
   * `.config/kdeglobals`
   * `.config/kcmfonts` (not found)

* Window Decorations
   * `.config/breezerc` (or your theme name)
   * `.config/kwinrc`

* Launch Feedback(cursor)
   * `.config/klaunchrc`

* Splash screen(after login, while loading desktop)
	* `.config/ksplashrc`

#### Workspace
* Desktop Effects
   * `.config/kwinrc`
   * `.config/kglobalshortcutsrc`

* Plasma Search
   * `.config/krunnerrc`
* File Search
   * `.config/baloofilerc`

#### Apps & Windows
* Task switcher
   * `.config/kglobalshortcutsrc`
   * `.config/kwinrc`

* Default Applications
   * `.config/mimeapps.list`

#### Input & Output
* Shortcuts
   * `.config/kglobalshortcutsrc`

* Accessibility
   * `.config/kaccessrc` (not found)
   
#### Dolphin
   * `.config/dolphinrc`
